Potrebno je imati installirano:
1.Python 3 (najbolje najnovija verzija ali sve python 3 verzije bi trebale raditi jer je ovo aplikacija napravljena u python verziji 3.9.4)
2.bcrypt (pip install bcrypt u command promptu ili visual studio code terminalu)
3.PIL (pip install Pillow u command promptu ili visual studio code terminalu)
4.matplotlib (pip install matplotlib u command promptu ili visual studio code terminalu)


Aplikacija se pokreće tako da kliknete dva puta na app.py ili korištenjem command prompte tako da navigiramo do PyFloraPosude datoteke i koristimo python app.py commandu. 
Nemojte pokušavati otvarati druge .py datoteke u ovom folderu jer neće prikazati ništa.


English instructions:

You need these things installed on your computer for this application to work correctly:
1.Python 3 (the most recent version would be prefered ,but any python 3 version should work because this is an application made in python version 3.9.4)
2.bcrypt (pip install bcrypt in the command prompt or visual studio code terminal)
3.PIL (pip install Pillow in the command prompt or visual studio code terminal)
4.matplotlib (pip install matplotlib in the command prompt or visual studio code terminal)

Application is launched by double clicking on the app.py file or using command prompt by navigating to the PyFloraPosude folder and then using the python app.py command. 
Do not try to launch other .py files in this folder cause they won't launch at all.
